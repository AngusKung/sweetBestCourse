      哪一學年度修課：99-1

      ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄)

         黃奎隆

      δ 課程大概內容

         Visual C# 2008
         1. An introduction to programming and C#
         2. Design a windows forms application
         3. Variable, data type, and operators
         4. Control statements and syntax
         5. Loops
         6. Array type
         7. Methods and events
         8. Structure types and class types
         9. Object-oriented and windows programming
         10.Graphics Device Interface
         11.Object Oriented Programming - Inheritance
         12.Polymorphism
         13.Directories and Files
         (大概就這些吧...比較有印象的)

      Ω 私心推薦指數(以五分計) ★★★★★
★★★
η 上課用書(影印講義或是指定教科書)

         Visual C# 2008 程式設計, 陳惠貞、陳俊榮, 學貫
         (暗紅色書皮)

      μ 上課方式(投影片、團體討論、老師教學風格)

         一個禮拜三節課
         第一節課通常就一直放投影片一直教
         中間都會正常上下課
         等所有投影片都教放完後
         會開始給大家看上課教的程式碼
         然後這也通常都是第二節課差不多結束了
         第三節課是所謂的lab time
         會指定練習題目要同學打出來
         打完就可以走了！

      σ 評分方式(給分甜嗎？是紮實分？)

         算紮實分吧！不調分，但也很好拿分

         Lab Assignments (25%)
         就是lab time所指定的實習，但也是變相的點名
         助教評分很鬆，就算程式碼打不出來
         但你有來有練習基本上就是送分！
         然後教授允許大家可以不到兩次

         homework (25%)
         一學期大概7次左右的作業
         但助教基本上也不會評太嚴格
         所以要拿分也滿簡單的
         有寫過計算機、提貨單跟賽車動畫等

         midterm exam (25%)
         期中考題目其實沒有太簡單
         比老師平常教得還要難一點
         但有兩部分的分數
         有大概45分的分數是上機寫程式
         55分則是選擇題+簡答題
         簡答題比較難一點，上機則比較好拿分數
         平均60左右

         final exam/ project (25%). 
         期末比期中還要難
         也沒有選擇題了
         考試題型一樣不過配分變50、50

         基本上有50分左右是可以全拿的
         然後考試兩次都有60分的話期末要80也不是問題

      ρ 考題型式、作業方式

         考試題型以及作業方式都在上面講了

      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)

         因為教室電腦有限所以不是每個人都簽
         一開始前面教得很簡單，進度也不快
         所以回去多加練習應該能很快跟上
         不過有基礎的人更好，學起來相對易上手
         老師不管你遲到也不太管你睡覺
         不過他看到會一直問怎麼大家這麼累之類的話
         但實習課你有出席就沒問題了

      Ψ 總結

         我覺得他上課的方式很普通
         一直放投影片講自己的、然後回頭問大家懂了嗎
         (總之還滿催眠的＝＝...zzZ)
         他的投影片有很多是從課本擷取
         所以不買課本只看投影片應該也還OK
         不過課本還是有其參考價值(請自己評估)
         我會給三顆星的原因是
         就真的是因為覺得上他課學不到什麼
         自己平常有練習比較重要
         不過這門課的好處是負擔不算太大
         對機械系學生來說，同時修這門課跟顧主科
         可能會比修電機系的計程來得輕鬆一點
