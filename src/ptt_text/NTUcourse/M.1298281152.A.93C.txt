※ 引述《TransAm (TRANS_AM)》之銘言：

 ◎課程名稱：
 　電磁學二
 ◎授課教師：
 　江簡富
 ◎學年度：
 　99-1
 ◎評分：
 　同樣 ★★★★★　

: ◎簡述：
:   還蠻推這班的,上課方式就是教授把推導寫在黑板上
:   然後我們一邊抄他一邊講解
:   推導從Maxwell's equations開始,熟練的話不太用硬記
:   雖然推導的方式,符號,順序都和課本不同
:   (也因為這樣,不建議缺課,自己看課本追進度相當麻煩)
:   不過聽完課,抄了筆記之後,基本上不太需要看課本
:   這對我是一個很大的優點,因為課本我看不太懂,又要看很久
:   除此之外,期中期末考前還放了幾次假
:   因為進度規劃有考量到放颱風假的可能
:   而實際上並沒有受到颱風的影響
:   作業的部分和統一教學勾的一樣
:   每個禮拜的進度回去就寫習題,然後下禮拜交,有時也會通融一下
:   這學期上課氣氛很不錯,下課很多同學會討論,或是到前面問教授
:   教授也會很認真的回答
:   總歸來說,是很值得一上的課
:   寫完發現自己寫的很不順= =
:   還請同班的多多補充

其實教授的重點都在觀念，他會用很簡單的物理現象先講起，

然後帶入一些例子，讓我們有一些圖像、實物做連結。

然後筆記確實一定要抄喔，而且最好跟同學對照確保抄對或是直接問教授，

否則抄錯的話你回去導半天都不知道在導什麼XD。

教授人非常好，即使像我有時候問題問不清楚，但是他仍然知道你不懂的點在哪！

這點讓我超佩服的。

我本身是以筆記為主課本為輔的在複習，上課幾乎不需要看課本，

你只要跟著教授的步驟走就好，教授在寫你就跟著抄，

抄好之後仔細聽，如果都有做到通常聽教授講完就懂了。

如果自己速度比教授慢一點可能需要下課去補問一下教授喔。


然後一些地方教授的教法跟課本不同（期末後半段），

不過我個人也是推薦聽教授的，課本的公式跟結果很多；

但是教授可以用一套方式講到底，很有系統的！

而且你聽完、自己回家複習重新導過，再回去看課本，

就會覺得課本幹嘛講這麼複雜......明明就很好懂的東西= =。

大致上是這樣啦，我個人覺得電磁二愈後面愈刺激，

也有真的在學電磁的感覺。

雖然第十章後半段以後都沒有提到了，可是教授說那才是電磁的精髓XDDD。

所以覺得有點可惜就是了。

以上，如果閱讀不便再煩請告知我囉！


P.S 發了文才忽然想起來XD，教授的課程網站有示範教學(Demo)，如果問完教授

還是有點迷糊，或是想加強觀念，都非常推薦去教授個人網頁

http://cc.ee.ntu.edu.tw/~jfkiang/index.html

左邊選單選Course→Electromagnetics→Demonstrations 裡面去看

通常教授會把每堂課該看哪個Demo附在Syllabus上面，所以也不必花時間找。

這個對我幫助真的滿大的，留下圖象印象就會比較好理解！
