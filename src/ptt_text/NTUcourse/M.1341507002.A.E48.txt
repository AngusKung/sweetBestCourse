      哪一學年度修課： 100-2

      ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄)
           何思特 Tobias Benjamin Hekster

      δ 課程大概內容
           演算法交易的邏輯與交易系統實作

      Ω 私心推薦指數(以五分計) ★★★★★
           有點難給, 沒進業界要高頻交易實作機會難得
           可是真要說其實這堂課也沒教什麼
           有興趣且有基本編程能力的話大概 4 吧

      η 上課用書(影印講義或是指定教科書)
           上課用投影片
           有參考書籍以及考試會從裡面出
           Algorithmic Trading and DMA: An introduction to
           direct access trading strategies
           http://www.amazon.com/dp/0956399207

           作者有在 Wilmott 討論區說因為私人因素, 書只有擺 Amazon 賣
           沒有代理. 學校有一本, 助教會幫大家請影印店印相關章節.

      μ 上課方式(投影片、團體討論、老師教學風格)
           因為是隔週上課, 進度有點零散
           前三次用投影片講交易有關的理論跟方法
           後面都是小組實作老師一組一組問有沒有問題

      σ 評分方式(給分甜嗎？是紮實分？)
           成績尚未公布

      ρ 考題型式、作業方式
           學期初說考試 50% 交易報告 50%
           學期末變成考試 60% 交易報告 40%
           我也不知道是哪個

           考試 40 題單選, 問跟上課教過還有參考書裡的內容
           主要是會看 limit-order book 跟會一些微結構的名詞
           難不難我覺得見仁見智, 財金背景跟這些材料很熟就不用太花時間
           如果沒有任何財務基礎可能就是全新的東西

           期末要交份報告, 內容關於小組的交易系統
           簡單說明 code 的內容跟最後 trading session 的結果
           解釋跟預期有什麼有什麼相同不同之類的
           本文兩頁 A4 並附上 code

      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)
           沒點名, 學期中適逢交易實驗室開幕, 有參加好像會加分
           基礎的話基本的財務商品要會, 上學期有門類似的課可能有幫助
           雖然那堂課最後好像都在教 Greeks, 沒用到也沒考到
           其他經濟學財務學知識可有可無, 最好有基本的編程能力
           學期初加簽熱烈, 幾十人抽四個, 可是第三周助教又說還有空位可以加選

      Ψ 總結
           我個人覺得這門課定位滿莫名奇妙的

           老師自己不是實作的人, 所以很多寫 code 的觀念不太會
           問他某某方法可不可行他大概會回 maybe, you can try
           用的程式系統是 Tango, manual 寫得之爛無法參透, 照打都可以錯
           跟一般程式語言不一樣的地方是交易是由 agents 下
           而 agents 有觸發條件要自己寫
           然後大部分會用到的語法也要由變數的變化去觸發
           比方說老師給的計算損益公式是錯的
           正確的寫法可以用 vector 紀錄買賣單
           並且在 Position 變化時觸發 vector 加值或拔掉值
           最後未實現的部位才能用市場價計算

           第四次上課之後其實就是去摸摸電腦寫寫程式
           模擬系統也很詭異, 因為是拿真實的資料所同步以有時會當掉
           但又可以寫個把市場上單都清光的程式, 所以不知是哪種模擬
           課程是輕鬆可是總覺得那有去跟沒去不是一樣...
           最後考試的東西他沒教的部分居多, 也是要自己看

           如果未來有興趣從事交易可以考慮修, 寫在履歷上應該有加分
           大可說是什麼獨步亞洲首創與香港大學同步的交易課程
           有個去德國的交易實習機會好像潛規則是這堂課跟上學期另一堂交易都要修
           比起德文 C1 還有用
