      哪一學年度修課：
ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄)
　　　　　　蔣明晃

      λ 開課系所與授課對象 (是否為必修或通識課 / 內容是否與某些背景相關) 
　　　　　　工商管理學系大二以上的選修
　　　　　　最重要的基礎是「矩陣」，雖然第一節課有簡要複習線性代數，
　　　　　　但還是知道矩陣怎麼運算就可以。

      δ 課程大概內容
　　　　　　02/18　Intorduction,Brief review of linear algebra
　　　　　　02/25　Linear Programming (Simplex Method)
　　　　　　03/04　Simplex Method(HW#1 due)
　　　　　　03/11　Sensitivity Analysis, Revised Simplex
　　　　　　03/18　Senstitvity Ananlysis
　　　　　　03/25　Senstitvity Ananlysis &amp; Duality
　　　　　　04/01　Transportation, Assignment &amp; Transshipment(HW#2 due)
　　　　　　04/08　Netwrok Modles
　　　　　　04/15　Midterm
　　　　　　04/22　Network Models
　　　　　　04/29　Interger Programming
　　　　　　05/06　Interger Programming　
　　　　　　05/13　Nonlinear Programming(HW#3 due)
　　　　　　05/20　Nonlinear Programming
　　　　　　05/27　Nonlinear Programming
　　　　　　06/03　Deterministic Dynamic Programming
　　　　　　06/10　Deterministic Dynamic Programming(HW#4 due)
　　　　　　06/17　Final Exam


      Ω 私心推薦指數(以五分計) ★★★★★
　　　　　　個人感受：　　　　　　　★★★★★
　　　　　　對於數學不好想混的人：　★★
　　　　　　對於數學好想混的人：　　★★★★
　　　　　　對於想學到東西的人：　　★★★★

　　　　　　詳細後述。

      η 上課用書(影印講義或是指定教科書)
　　　　　　Winston, W.L &amp; M. Venkataramanan, Introduction to Mathematical
　　　　　　Programming, 4th edition, Thomson Learning

　　　　　　老師有講義，主要取材都是課本。念通講義其實就不必看課本。

      μ 上課方式(投影片、團體討論、老師教學風格)

　　　　　　老師直接講課，不過老師其實講話步調很慢。所以進度一直有些許延遲，
　　　　　　不知道為什麼最後居然還是全部都上得完，這一切都是謎。

　　　　　　老師的投影片把重點都有納進來，但是由於這門課的重點在於學習要如何讓
　　　　　　一個規劃問題得到最佳解，因此我們主要要學習「一套方法」（也就是某種
　　　　　　算術），將各種方法都能熟悉並應用於題目中即可。不過算術過程往往繁雜
　　　　　　，老師不會在投影片上描述得非常精細，因此上課勤做筆記是上策。不然單
　　　　　　是從從投影片中的Step去理解到底該做甚麼動作來解題還滿困難的。

      σ 評分方式(給分甜嗎？是紮實分？)
　　　　　　甜，但應該算是扎實的。
　　　　　　於下段描述

      ρ 考題型式、作業方式
　　　　　　老師在考前大致上會描述自己會考的題型與範圍，一律採open book/notes
　　　　　　但不可以使用筆電。評分上寫期中期末各占30%，作業占40%(每次10分)

　　　　　　作業就是上課提到的方法，應用到題目中的case。我個人是覺得作業還滿難
　　　　　　寫的，不過期中期末考比較簡單。不過主要原因是作業是給現實狀況，需要
　　　　　　親自列式(我往往敗在這裡)，但期中期末都是給好式子，就可以讓妳去解題
　　　　　　了。

　　　　　　原PO期中93，期末90，學期成績為91。班平均84.35，標準差為13.64
　　　　　　其實班上還滿多人的最後成績比我高很多，
　　　　　　四十五人修課，五人停修，三個人被當。

      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)
　　　　　　出席率不重要，基礎就是矩陣，還有一些些高中的數學。(線性規劃)
　　　　　　修課的人感覺不多，所以老師有加簽我(我第一周沒到)

　　　　　　另外我覺得數學底子好其實還滿占優勢的，因為其實期中成績知道後我就
　　　　　　都帶筆電看FB + 打網誌，筆記也做得比較少。原本想說會炸掉，最後其實
　　　　　　還不錯。

      Ψ 總結
　　　　　　對於數學不錯的人來說挺涼的。
　　　　　　不過其中的內容確實挺吸引人，是門好課。
　　　　　　作業量不會很大，但需要將課堂上的東西融會貫通，
　　　　　　考試相當容易，按照講義上的重點套用上去幾乎都解得出來。

　　　　　　作為三學分的課，考試三小時，大概90-120分鐘就可以寫完交卷了。
