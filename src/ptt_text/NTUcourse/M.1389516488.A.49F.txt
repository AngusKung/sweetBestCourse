       哪一學年度修課：

                 101-2

      ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄)

                 蔣明晃

      λ 開課系所與授課對象 (是否為必修或通識課 / 內容是否與某些背景相關) 

                 工商管理學系 商學研究所 選修

      δ 課程大概內容

                 02/18  Intorduction,Brief review of linear algebra
　　　　　　     02/25　Linear Programming
     　　　　　　03/04　Simplex Method
　　     　　　　03/11　Sensitivity Analysis, Revised Simplex
　　　　     　　03/18　Senstitvity Ananlysis
     　　　　　　03/25　Senstitvity Ananlysis &amp; Duality
　     　　　　　04/01　Transportation, Assignment &amp; Transshipment
　　     　　　　04/08　Netwrok Models
     　　　　　　04/15　Midterm
　     　　　　　04/22　Network Models
     　　　　　　04/29　Interger Programming
　     　　　　　05/06　Interger Programming　
     　　　　　　05/13　Nonlinear Programming
     　　　　　　05/20　Nonlinear Programming
     　　　　　　05/27　Nonlinear Programming
     　　　　　　06/03　Deterministic Dynamic Programming
　     　　　　　06/10　Deterministic Dynamic Programming
     　　　　　　06/17　Final Exam

      Ω 私心推薦指數(以五分計) ★★★★★

                 ★★★★★

      η 上課用書(影印講義或是指定教科書)

                 有投影片，但課本很詳細很棒，建議買
                 Winston, Operations Research, 4th edition, Thomson Learning

      μ 上課方式(投影片、團體討論、老師教學風格)

                 投影片上課，但因為一個班人數不多，所以有些比較認真的同學很敢發
                 問。上課進度在期中考前落後不少，可是期末考卻謎一般的全部預期進
                 度都教完了...。

                 這門課的先修基礎是管理科學模式，但我是同學期修。基本上管科模重
                 視實用，最適化重視背後的理論，我覺得同一個學期修讓我很有收穫，
                 也更能了解規劃求解的背後意涵。

                 老師上課會用一些板書但因為矩陣一個都不算小，所以要坐前面才抄得
                 到，算是小缺點。

      σ 評分方式(給分甜嗎？是紮實分？)

                 期中考期末考各占30%，作業四次40%。

      ρ 考題型式、作業方式

                 考試試五到六題的計算題，都不難，而且可以開書所以不用很擔心。
                 大概50人修課，5~6個人停修，最後的成績表是3個人被當。
                （最後有調分）A+的人非常多，對我來說覺得是扎實偏甜。

                 作業是課本習題，和考試很類似，但我覺得比考試難，建議作業認真做
                 ，可以當作檢視你上課懂不懂的基準。

      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)

                 基礎是管理科學模式和線性代數，但不會有很大影響。

                 加簽都會簽。

      Ψ 總結

                 這門課須有有一點點線性代數和多項式微分的基礎，但第一堂課老師會
                 複習，老實說用到的觀念並不難。沒有來上課會讓你看課本看到想哭，
                 老師講解的很好，而且有些課本沒有特別註明的小細節老師會上課題點
                 ，所以建議一定要出席。

                 先修是管理科學模式，但其實沒有很大影響，兩門課偏重的東西差很多
                 ，如果只是想要會用Excel的話可以不必來修最適化，念好管科模即可
                 。
