      哪一學年度修課： 102

      ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄) 陳維玲

      λ 開課系所與授課對象 (是否為必修或通識課 / 內容是否與某些背景相關) 
　　　　　沒限定，老師說會全簽

      δ 課程大概內容　
　　　　　　　　　
　　　上學期：
　　　les pronoms complements directs et indirects ;
　　　place de l’adjectif ; le passe compose ; l’
　　　imparfait ; la negation ; l’imperatif negatif ; le
　　　futur simple ; futur proche ; place des adverbes
　　　下學期
　　　Les pronoms possessifs, les relatifs,
　　　les indefinis, le passe recent,
　　　le style indirect, le conditionnel, subjonctif,
　　　verbes impersonnels, la voix passive,
　　　les comparatifs, les superlatifs ...

      Ω 私心推薦指數(以五分計) ★★★★★
　　　　　★★★★★ 五顆
      η 上課用書(影印講義或是指定教科書)
　　　　　initial deux et le cahier
　　　　　老師會自己編講義傳到網上，要先印好上課一邊看講義一邊看課本
　　　　　ＰＳ老師的講義編得很好！！
      μ 上課方式(投影片、團體討論、老師教學風格)
　　　　　主要是老師口頭授課，搭配講譯課本和習作
           講義是課本文法和會話的整理
           講義上老師還會補充很多法國文化的東西

　　　　　　會點同學回答問題或念課文，老師也趁你念課文的時候會糾正你的發音
　　　　　　課文分成兩部分
　　　　　　第一個部分是老師會直接講解的
　　　　　　第二個部分老師會請大家回家先查單字翻譯
　　　　　　下一次上課再點人來翻

           考完試的下一堂課老師通常會放法文電影或影集

           助教一學期會做兩次介紹和法國文化的報告，滿有趣也很輕鬆

      σ 評分方式(給分甜嗎？是紮實分？)
　　　　　　一學期有四次考試，通常會有聽寫　文法　法翻中
　　　　　　聽寫都是課本或習作裡的句子，習作每一課老師會勾兩大題是要考聽寫的
　　　　　　，即使有變化也都差不多
　　　　　　所以這一部分有唸一定有分，很容易拿分

　　　　　　文法有時候會比較難判斷，但也還ＯＫ

　　　　　　法翻中的句子也都是課本和講義裡的，而且老師會特別強調某些句子，
　　　　　　所以很容易知道要考什麼，不難準備
      ρ 考題型式、作業方式
　　　　　　除了剛剛講的筆試之外，一個學期會有兩次演戲
　　　　　　自己找兩三個人一組
　　　　　　編一個跟課本內容有關的對話
　　　　　　大約兩三分鐘的長度就好
　　　　　　文法有錯的話老師會糾正你
　　　　　　應該不會扣分
　　　　　　除非發音真的錯得太離譜
　　　　　　這部分評分不清楚
　　　　　　但感覺老師人很好，不會在這裡刁難

　　　　　　作業的話就是要先回家查好課文單字
　　　　　　下一次上課老師會點人翻譯
　　　　　　還要寫習作
　　　　　　老師不會檢查
　　　　　　只要他點到你你會回答就可以

      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)
　　　　　　老師應該滿重視學習態度的
　　　　　　但如果你不小心遲到幾次
　　　　　　只要不是太多次
　　　　　　然後來了有認真上課
　　　　　　老師應該不會介意
　　　　　　
　　　　　　不管什麼系幾年級都可以選這門課
　　　　　　老師說他會全部加簽
　　　　　　沒修過法文一也可以簽法文二
　　　　　　我真的超感謝老師這一點
　　　　　　因為我之前在外面學過法文，沒有修法文一
　　　　　　別的老師都不給加簽或旁聽
　　　　　　很感謝老師願意讓我修這門課　
      Ψ 總結
　　　　　　老師是個很熱心的人
　　　　　　平常如果你幾練習了寫法文句子只要寄給他他都會認真幫你改
　　　　　　我修完一年課
　　　　　　連暑假的時候我寫法文老師也都會幫我改
　　　　　　真的進步很多
　　　　　　而且每次上課都會被點到兩三次
　　　　　　感覺口說也有很大的改善
　　　　　　推薦給想學法文的人來試試看！
