      哪一學年度修課：103-1

      ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄)
林宗男

      λ 開課系所與授課對象 (是否為必修或通識課 / 內容是否與某些背景相關) 
電機系 選修

      δ 課程大概內容
Overview
Application Layer
Transport Layer
Network Layer
Link Layer
Mobile and Wireless Networking

      Ω 私心推薦指數(以五分計)
原本要給  ★★★★★
可是看到成績後很難過 只能給 ★★★★
η 上課用書(影印講義或是指定教科書)
James F. Kurose &amp; Keith W. Ross; Computer Networking: A Top-Down Approach
Featuring the Internet, 6th Edition; Addison Wesley

但是老師會有投影片，並會上傳至Ceiba

      μ 上課方式(投影片、團體討論、老師教學風格)
基本上老師都會用投影片上課
有時候也會把投影片關掉，然後用寫的方式教學
老師有時候會點同學回答一些小問題
例如你的看法如何啊之類的
然後宗男老師上課真的很有趣
會用一些很神奇的例子來譬喻網路
例如什麼菩薩和信男信女之類的
聽了真的會覺得很有趣XDDD
雖然老師上課有點跳來跳去(就是沒有跟投影片的順序一樣)
然後有時候連兩個禮拜都會講到同一個東西講很多遍
但大致上我很喜歡老師的上課風格
真的會讓心情很好:D

      σ 評分方式(給分甜嗎？是紮實分？)
Midterm exam 30%,
Final exam 30%,
Homework assignment 30%,
Class participation 10%

作業有六份，前三分有公布成績，我都拿A+
後三份沒公布，但我自認為我寫得還不錯

期中平均6X，我拿70
期末平均7X，我拿6X

學期成績 only B QQ

應該是完全沒有調分吧呵呵呵呵

期中期末考總分超過一百
拿九十以上的人很多，大約三分之一吧
但是我真的不是個會考試的人
所以成績好爛噢噢噢噢噢
讀這麼久，學期成績這樣真的蠻難過的呵呵呵呵呵
可是也沒辦法誰叫我考試考這麼爛呢
虧我還蠻喜歡這堂課的，可是看到成績還是很難過:P

BTW
我同學期中90 期末95
作業成績大約A
上課常常睡覺
但還是拿了A+
覺得難過QQ

      ρ 考題型式、作業方式
【作業】
其實作業講解都不是很清楚
不知道是不是因為沒有助教還是怎樣@@
有時候到交作業隔天，老師才突然說作業可以再延

作業一是traceroute
會有忘記是一次還是兩次的上機讓我們練習+問問題

作業二要用wireshark做實驗
2.1 交一份PPT檔
2.2 交一份technical report
老師會給我們範例，以及希望我們用Latex寫
我覺得這份作業讓我學到蠻多寫report的技巧

作業三跟封包有關
3.1 再次用wireshark研究 window的變化量，一樣寫成 technical report
3.2 老師會給封包傳送程式，然後我們再修改成可以傳送檔案

作業四是寫Dijkstra的程式

【考試】
考試的部分啊......
因為我真的不太會考試
所以考完我也不太會解釋考試題型
就去看其他人寫的評價吧:P

      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)
1. 出席
老師會點人回答問題，可能算另類點名吧

2. 外系選修
還蠻多外系選修的OAO
不過最好有修過計算機程式還有計算機概論
如果有學過資料結構更好
因為後兩個作業要寫程式
而且會用到Linux的作業系統
沒學過但想修這門課的人最好先去研究一下

3. 老師個性
老師很可愛:P
下課還會放天空之城還是哪首宮崎駿的歌(?)

4. 加簽
不太清楚，但應該都是會給簽吧

5. 遲到
很多人遲到
甚至有人第二節、第三節才來
應該是還好吧@@
