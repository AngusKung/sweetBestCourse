      哪一學年度修課： 103-2

      ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄)

         陳柏華  助理教授

      λ 開課系所與授課對象 (是否為必修或通識課 / 內容是否與某些背景相關) 

         土木系大一必修 (不限本系生，外系加簽也可以)

      δ 課程大概內容

         第 1週    228連假(停課)
         第 2週    Ch1   : Introduction to Java
         第 3週    Ch2   : Elementary Programming
                   Ch3   : Introduction to Selection
         第 4週    Ch3-4 : Selection , Math Functions , Characters and Strings
         第 5週    Ch5   : Loops
                   Quiz 1
         第 6週    Ch6   : Methods
         第 7週    春假(停課)
         第 8週    Ch7   : Singal Dimansional Arrays
                   Quiz2
         第 9週    Ch8   : Multi-Dimansional Arrays
         第10週    Midterm
         第11週    Ch9   : Introductions to Class and Objects
         第12週    Ch10  : More on Class and Objects
         第13週    Ch9-10 延伸
                   Quiz3
         第14週    Ch11  : Inheritance and Polymorphism
         第15週    Ch12  : Inheritance and Polymorphism Ⅱ , Abstract Class and
                           Interface
         第16週    Ch14  : JavaFX Basics
                   Quiz4
         第17週    Ch15  : JavaFX and Graphical User Interface
         第18週    Final Exam
Ω 私心推薦指數(以五分計) ★★★★★

         初學程式語言者 ★★★★

                                ／ 想輕鬆拿高分的 ★★★★★
         已學過其他程式語言者，∣
                                ＼ 想學java的 ★★
(主要概念都差不多，只有JavaFX那邊比較新，其實可以自學~)
η 上課用書(影印講義或是指定教科書)

         Y. Danial Liang (2012), Introduction to Java Programming.Comprehensive
         Version (9th Edition), Practice Hall

        上課前老師會把講義(從課本節錄)放在ceiba上  沒買課本的可以印
        期中(末)考前會把有解答的講義放上ceiba

      μ 上課方式(投影片、團體討論、老師教學風格)

         課堂前半部(約2 ~ 2.5節課的時間)，老師會講解上課內容，剩下的時間就是lab
         實作，題目跟上課內容相關，做完就可以提早走喔 ~

         老師的教學風格頗有趣，不時會講些冷笑話(他說是為了避免我們睡著...)。印象
         最深的就是之前地震，還叫我們趕快趴到桌子底下 !? 結果全班都在躲，呵呵~

      σ 評分方式(給分甜嗎？是紮實分？)

         土木系計程的3個班標準都一樣

         Lab          20%
         Quiz         40%
         Midterm      15%
         Final Exam   25%

         給分寬鬆度要看這次改的是哪個助教@@ ，雖然有評分標準，但要給到多少是助教
         的權限，像期中考的助教改的較鬆，基本上能給都盡量給，但是期末就很嚴了QQ

         不過這學期老師其實有變相加分，期中滿分150可以補其他考差的地方，到最後平
         均下來分數應該不會太難看(有在讀的話拉...)

         最後打等第基本上就是看平均了，但是如果你很接近60的話，老師還是會給你C-
         ，真的被當的人數其實很少

      ρ 考題型式、作業方式

         考題就是標準的計程考法：

         (1)紙筆測驗：就是考一些重要的基本觀念，裡面可能會有一些陷阱(i和j會亂跳
                      )，要仔細看題目喔~

         (2)上機考  ：老師會發題目下來，就自己想辦法在時間內解決了~
                      這時老師會切斷網路，直到考完要大家上傳ceiba才會開

         上機考的題目有機會跟作業的題目類似(甚至一模一樣)，考試時可以帶紙本資料
         (自己印的code 、 講義和課本等)作為參考，但不能用任和電子產品，所以考前
         記得要去印資料喔 \^。^/

         期中考的分數有60分跟作業一模一樣，小考就只有類似的題目，但期末跟作業有
         關的比例就很低，而且都出在最後一小題，如果你每一題的(a)就不會了，那就
         算你知道(c)怎麼做也沒用啊~~  T.T

         作業跟上課要交的Lab不同，作業不須繳交但題目通常比較難，老師也不會把解答
         po在ceiba上，要自己找老師或助教約時間看，但是他們的office hour常會卡必
         修，所以真的去找的其實不多，有問題就密fb討論就好 ~

         每次期中(末)考前，大家幾乎都會分享作業的答案，就算完全不會靠作業的
         carry還是能讓你過(只是成績就不要要求太多)

         原po Lab 平均 99 ，Quiz 平均 86 ， Midterm 149 ， Final 89
         最後是拿A+  給大家參考囉~

      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)

         老師原則上都不點名，如果都會其實可以不用去(Lab記得準時交就好)，有幾次
         上課時人數真的少的離譜才有點，但老師說那只是加分性質，沒點到不會扣分，
         越到學期末每次準時上課的人就越少，很多人都遲到 (早9應該還好吧 ~ )

         土木系原本的重心本來就不在計程，所以沒基礎不會有問題
         (系上有學過的應該也沒幾個)

         受到教室電腦數量的限制，加簽原則上就是看教室還有幾個位子，簽滿就沒辦法
         了。這學期助教是用抽學生證的方式選人，選不到也可以去其他2個班試試看

      Ψ 總結

         這個老師聽說是三個班中最好的，上課有聽不懂的都能立刻舉手發問，老師也都
         會盡力解答，Lab有問題也都能有耐心的回復，甚至課後有問題還能用fb問，這樣
         的老師怎麼可能不推！只要有心想學好這門課，在討論間可以學到不少喔 ~

         另外，上課也能帶自己的筆電去，這樣老師邊上課你可以邊打Lab，這樣就不太會
         耽誤到下課時間(但是大多數人都用筆電做自己的事  呵呵~)。但是計程的難度其
         實因人而異，有些邏輯比較強的可能就很容易想出解法，但有些人就很容易卡住
         ，不過總體而言還是推薦有興趣的來修，畢竟這麼好的老師真的不多 ~
