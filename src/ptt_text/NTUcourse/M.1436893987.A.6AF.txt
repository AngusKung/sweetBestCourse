      哪一學年度修課：103-2

      ψ 授課教師 (若為多人合授請寫開課教師，以方便收錄)

         林宗男

      λ 開課系所與授課對象 (是否為必修或通識課 / 內容是否與某些背景相關) 

         電機所、電信所

      δ 課程大概內容

         1.Introduction
         2.The Physical Layer
         3.The Data Link Layer
         4.The Medium Access
         5.Network Layer
         6.The Transport Layer
         7.The Appoication Layer

      Ω 私心推薦指數(以五分計) ★★★★★

         修過電腦網路導論後想輕鬆拿學分★★★★★
         但是還是要記得交作業

         想要學東西★
         自己唸好像比較有效率（？

      η 上課用書(影印講義或是指定教科書)

         Andrew S.Tanenbaum“Computer Networks”Third Edition 1996 Prentice-
         Hall International Inc.
         已經出到第五版了但課程網上還是放第三版

      μ 上課方式(投影片、團體討論、老師教學風格)

         投影片、板書（如果抄起來有用的話？）

      σ 評分方式(給分甜嗎？是紮實分？)

         期初：Paper review 10% 有四篇
               Homework     30% 不知道有幾次作業
               Midterm      30% 沒有考古題
               Final        30% 同上

         期末：Paper review  8% 兩篇，一篇4%
               Homework     32%
               Midterm      30%
               Final        30%

         調分後拿B-
         一半以上的人至少A-
         三十九個人中應該不到五個被當

      ρ 考題型式、作業方式

         作業：
               Paper review (8%)

                 要你看兩篇 Paper，寫個評論之類的
                 第一次作業後才丟出範例給我們參考

               Trace route map (2%)

                 顧名思義就是那樣，滿簡單的

               Python Lab 1~6 (6%)

                 在 https://www.codecademy.com 上面註冊一個帳號
                 然後在上面學Python，就這樣

               SDN Lab 1~4 (18~24%)

                 有四個作業，簡單來說就是
                 1.用 mininet 去模擬一些網路
                 2.然後用 wireshark 去觀察
                 3.有時用 Python 去修改現有的 code
                 4.再用 wireshark 去觀察
                 5.一定要寫 Report

               Python Lab 7 (0~6%)

                 用 Python 寫一個 Shortest Path Algorithm
                 會依照效率去給分，也是滿簡單的

      ω 其它(是否注重出席率？如果為外系選修，需先有什麼基礎較好嗎？老師個性？
加簽習慣？嚴禁遲到等…)

         老師第一堂課說要修過"電腦導論"（你沒看錯，就是這四個字）
         我知道臺大近十年絕對沒有人開過這門課
         所以猜測老師所說的應該是"電腦網路導論"
         出席率完全不重要
         上課不能吃東西
         一定要會寫程式（不一定要先學 Python 也沒關係）
         最好要熟linux系統（比較方便）
         最好修過電腦網路導論（電機系）或是計算機網路（資工系）
         課程加簽類別是第一類=&gt;網路加選（保證隨便選隨便上）

      Ψ 總結

         抱怨：

           1.SDN Lab 1~4 和 Python Lab 7 五個作業中選四個高分的做成績計算
             ，這件事在學期結束後才公告

           2.這門課表面上說是電網導的進階課程，但即使我沒修過電網導也知道
             這門課就是電網導

           3.投影片相關 a.期中考後的投影片太晚放上 ceiba
                        b.投影片看起來有點隨便，很多排版都亂排
                        c.會跳過很多投影片不講（那你幹麻放上來？）

           4.說真的老師的口條不太好，雖然我都知道老師在說什麼，但是老師講
             話不是很順暢，要理解完整內容可能要在腦中先 compile 過再 link
             才能 output

           5.Paper review 一開始說四篇，最後變兩篇，很多人應該會想這不是很
             好嗎？可惜我第一篇沒交，所以比例被加重了
        　　（當然這有一半是我的問題）

           6.作業和上課的內容連結不夠強，不清楚作業的目的想給我們帶來什麼

           7.考試雖然就是上課的內容，不過沒有練習過說真的不好寫

         學到的東西：

           1.Python

           2.看原文書的速度加快（都自己唸啊！）

           3.不要修這門課（誠心建議）
